### 連想

worse still は、worse と still を別々に丸暗記するより、2語以上がまとまって1つの場面を作る表現として覚えると定着しやすいです
このイメージから、`いっそう悪いことに` という意味につながる。
補足として、反対表現は better still [yet]「さらによいことに」 という点も一緒に覚えておくとよい。

### 類義語
- worse still
  - 対象の意味は「いっそう悪いことに」。この熟語特有の語順・前置詞まで含めて覚える
- worse yet
  - 同じ項目で扱われる別形。意味は近いが、使える形をまとめて覚える
- to make matters things worse
  - 意味は近いが、後ろに続く語や文型が異なることがある
- what is worse
  - 近い意味を表す表現。細かな文型や響きの違いに注意する
