### 連想

set about ~ は「〜の周りに身を置いて取りかかる」イメージ。これからする作業に向かって動き始める ⇒ 〜を始める、となる。

### 類義語
- set about
  - 仕事や作業に取りかかることを表す
  - 後ろは名詞や動名詞
- begin
  - 「始める」
  - 最も基本的
- start
  - 「始める」
  - begin より口語的で幅広い
- set out to do
  - 「〜し始める、〜することに着手する」
  - 目的を持って始める感じ
