### 連想

number は「数」。a number of ~ は「ある数の〜」＝ ひとまとまりの数がある ⇒ いくつもの〜、かなり多くの〜、という流れで覚える。

ポイントは a が付いていること。a number of ~ は「いくつもの〜」だが、the number of ~ は「〜の数」そのものを指す。

### 類義語
- a number of
  - 可算名詞に使い、「いくつもの」「かなり多くの」という意味
  - a large number of とすると「非常に多くの」、a small number of とすると「少数の」
- many
  - 可算名詞に使う一般的な「多くの」
  - a number of より直接的で、説明文や会話でも幅広く使える
- several
  - 「いくつかの」で、a number of より少なめの印象
  - 「かなり多い」までは含みにくい
- the number of
  - 「〜の数」という意味で、a number of とは別物
  - The number of students is increasing. のように、主語は number になる
