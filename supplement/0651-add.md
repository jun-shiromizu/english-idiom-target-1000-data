### 連想

for sure は、for と sure を別々に丸暗記するより、2語以上がまとまって1つの場面を作る表現として覚えると定着しやすいです
このイメージから、`確かに` という意味につながる。
補足として、副詞句の場合は surely, certainly。That's for sure [certain]. は「それは確かだ」の意味で、for sure [certain] は形容詞句 という点も一緒に覚えておくとよい。

### 類義語
- for sure
  - 対象の意味は「確かに」。この熟語特有の語順・前置詞まで含めて覚える
- for certain
  - 同じ項目で扱われる別形。意味は近いが、使える形をまとめて覚える
- surely
  - 1語で言える近い表現。文脈によって置き換えやすい
- certainly
  - 1語で言える近い表現。文脈によって置き換えやすい
