### 連想

only too は、only と too を別々に丸暗記するより、2語以上がまとまって1つの場面を作る表現として覚えると定着しやすいです
このイメージから、`(glad, happy, pleased などの前で)とても、大いに；遺憾[残念]ながら` という意味につながる。
複数の意味がある場合も、中心になる構文・前置詞・場面を押さえると、文脈ごとの意味を選びやすい。
補足として、①②は all too と同義 という点も一緒に覚えておくとよい。

### 類義語
- only too
  - 対象の意味は「(glad, happy, pleased などの前で)とても、大いに；遺憾[残念]ながら」。この熟語特有の語順・前置詞まで含めて覚える
- all too
  - 近い意味を表す表現。細かな文型や響きの違いに注意する
