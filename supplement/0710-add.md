### 連想

if only ... は、if と only を別々に丸暗記するより、2語以上がまとまって1つの場面を作る表現として覚えると定着しやすいです
このイメージから、`…でありさえすれば；(現在・未来の願望を表して)…でさえあればなあ` という意味につながる。
複数の意味がある場合も、中心になる構文・前置詞・場面を押さえると、文脈ごとの意味を選びやすい。
補足として、if only because ... は「…だけが理由だとしても」の意味。if only の節だけで独立して用いることが多い。節の中は仮定法の動詞を用いる という点も一緒に覚えておくとよい。

### 類義語
- if only ...
  - 対象の意味は「…でありさえすれば；(現在・未来の願望を表して)…でさえあればなあ」。この熟語特有の語順・前置詞まで含めて覚える
- only if ...
  - 近い意味を表す表現。細かな文型や響きの違いに注意する
- I wish ...
  - 近い意味を表す表現。細かな文型や響きの違いに注意する
