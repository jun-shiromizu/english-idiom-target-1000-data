### 連想

beyond description は、beyond と description を別々に丸暗記するより、2語以上がまとまって1つの場面を作る表現として覚えると定着しやすいです
このイメージから、`言葉では表現できないほどの[に]` という意味につながる。

### 類義語
- beyond description
  - 対象の意味は「言葉では表現できないほどの[に]」。この熟語特有の語順・前置詞まで含めて覚える
- beyond recognition
  - 近い意味を表す表現。細かな文型や響きの違いに注意する
- beyond (a) doubt question
  - 意味は近いが、後ろに続く語や文型が異なることがある
