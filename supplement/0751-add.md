### 連想

go so far as to do は、as で対応する形を示し、so で同じ関係を受ける比較・対応の構文です
このイメージから、`…しさえする` という意味につながる。
補足として、go so far as doing に加えて、go as far as to do [doing] の形もある という点も一緒に覚えておくとよい。

### 類義語
- go so far as to do
  - 対象の意味は「…しさえする」。この熟語特有の語順・前置詞まで含めて覚える
- より直接的な基本表現
  - 日本語訳に近い意味を1語や短い表現で言い換える場合に使う。試験では熟語の形そのものを優先して覚える
- 文脈に応じた言い換え
  - 同じ日本語訳でも、対象・文体・前後関係によって自然な英語表現が変わる
